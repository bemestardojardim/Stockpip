# 📟 Vault-Tec Pack Management System

Sistema de estoque estilo **Fallout** para gerenciar faltas da seção **Packs** (18 cervejas). Interface tipo Pip-Boy 3000, tutorial interativo, drag-and-drop, edição rápida e temas ☢️. Ideal para bares e estoques.

🔧 **Funcionalidades:** Edição sequencial, modo gerenciar, reset setor, persistência local.

🖼️ Use `pipboy.png` (32x32) na mesma pasta.
